🎬 Movie Recommendation System

A professional Content-Based Movie Recommendation System developed using Machine Learning, Natural Language Processing (NLP), and Streamlit.
The application recommends movies based on similarity in genres, keywords, cast, crew, and movie descriptions.

The system leverages CountVectorizer and Cosine Similarity to generate accurate movie recommendations and integrates the TMDB API to display real-time movie posters.

🚀 Features
Content-Based Movie Recommendation Engine
Personalized movie suggestions
Real-time movie poster integration using TMDB API
Interactive and responsive Streamlit web interface
Fast recommendation generation using similarity matrix
Netflix-inspired dark theme UI
